# Organizaci-n-Industrial-Cap-tulo-2
Página web interactiva que analiza el caso de Dunkin’ Donuts aplicando conceptos de Organización Industrial como costos de transacción, integración vertical y límites de la empresa. Incluye una visualización del punto óptimo y fue desarrollada con HTML, CSS y JavaScript para facilitar la comprensión de la teoría de forma práctica.
